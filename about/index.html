关于本站

欢迎来到我的博客！这里是我记录技术成长、分享学习心得、探讨编程实践的地方。

技术栈

本博客采用现代化的全栈技术构建：

  • 后端：Spring Boot 3 — 利用 Java 生态的强大能力，提供稳定高效的 API 服务
  • 前端框架：Vue 3 — 响应式、组件化的现代前端开发体验
  • 服务端渲染：Nuxt 3 — 优秀的 SEO 支持与首屏加载性能

选择这套技术栈，是希望在实际项目中探索和实践当前主流的开发模式，同时为访客提供流畅的浏览体验。
