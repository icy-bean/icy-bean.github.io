保姆级教程：手把手教你如何使用FlClash科学上网

2026年3月8日 11:36 阅读约 4 分钟实用工具教程 25 次浏览 4 个赞 0 条评论
工具 技术
摘要
FlClash是一款基于Flutter开发的跨平台Clash图形客户端，用于解决访问GitHub等国外网站时遇到的网络问题。它通过规则匹配智能分流，国内网站直连，国外网站走代理。使用前需关闭其他加速器软件，并获取代理订阅链接。本文提供了从注册购买订阅、下载安装客户端到配置FlClash的详细步骤，帮助用户实现科学上网。

一、什么是 FlClash？

当今网络环境下，当你访问github等国外网站时，有时会一直加载，无法正常访问。此时就需要网络代理来解决这一问题，FlClash 使用Flutter框架开发，是Clash的一个现代化、跨平台的图形客户端，注重易用性和多设备一致性体验，在本文使用FlClash来作为跨平台代理客户端。

工作原理示意图

┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│   你的设备   │ ───→ │FlClash客户端│ ───→ │   目标网站   │
└─────────────┘      └─────────────┘      └─────────────┘
                          │
                          ├─── 规则匹配 ──→ 直连（国内网站）
                          │
                          └─── 规则匹配 ──→ 代理（国外网站）

二、准备工作

在开始之前，请确认以下事项：

关闭加速器软件：停止雷神 UU...等等一系列VPN加速器软件
确认设备系统：根据操作系统选择对应的客户端
稳定的网络：确保网络连接正常

获取代理订阅教程

⚠️ 重要提醒：订阅链接属于私密信息，请勿公开分享或在不安全的网站上使用。

第一步：访问魔戒官网登录并注册

在浏览器地址栏输入：mojie.app/register?aff=dhbDn3ZG (邀请码：dhbDn3ZG)回车进入魔戒官网，注册一个你的账号并登录。

第二步购买订阅

点击侧边栏的购买订阅

由于代理需要中转站，也就是俗称机场资源，所以使用任何代理都需要付费的，而魔戒是高性价比的不限时的代理服务提供商，进入商店选购一款适合你的套餐，一般来说，购买14.9的130G套餐足够你用很久了。

下单完成支付后在侧边栏回到首页，点击一键订阅，复制弹出对话框的链接，这就是我们等下会用到的代理订阅链接。

订阅链接格式示例：
https://example.com/api/v1/client/subscribe?token=xxxxxxxxx

四、配置 FlClash

步骤 1：启动客户端

安装完成后，启动 FlClash，你会在任务栏（系统托盘）看到一个三条蓝色斜杠图标。

步骤 2：导入订阅链接

  1. 点击主界面的 配置 （文件夹样式）选项卡
  2. 点击添加配置按钮
  3. 选择通过URL获取配置文件，
  4. 把刚刚从魔戒机场获取到的订阅代理链接粘贴进去
  5. 点击右侧 提交 按钮下载配置
  6. 下载成功后，你会看到成功配置的魔戒代理

步骤 3：开启代理

在侧边栏返回首页，将系统代理选项打开，最后点击右下角的启动按钮启动代理服务。

步骤 4：测试连接

至此，已经大功告成了，现在可以尝试着打开浏览器访问以下任一网站测试：

# 测试 Google
https://www.google.com

# 测试 YouTube
https://www.youtube.com

# 查看 IP 地址和地理位置
https://ip.sb

如果能正常访问，说明配置成功！

五、出站模式说明

在首页中有一个出站模式的选项卡，一般使用默认的规则模式即可，会智能分配出站流量和选择最佳代理节点。

模式说明适用场景
Rule（规则）根据规则智能分流日常使用（推荐）
Global（全局）所有流量走代理特殊需求场景
Direct（直连）所有流量直连国内访问

六、常见问题（FAQ）

❓ 连接成功但无法访问外网？

排查步骤：

  1. 检查代理模式是否为 Rule 或 Global
  2. 确认已选择可用的代理节点
  3. 尝试更换其他节点
  4. 更新订阅获取最新节点列表

❓ 系统代理开关无法打开？

可能原因与解决：

原因解决方案
权限不足以管理员身份运行 FlClash
端口冲突检查是否有其他代理软件占用端口
软件冲突暂时关闭其他 VPN/代理软件

❓ 电脑开机后 FlClash 不能自动启动？

解决方法：在 工具 应用程序设置中开启 自启动。

❓ 如何更新订阅节点？

在 Profiles 页面，找到配置文件名称右侧的更多按钮中点击同步，点击即可更新节点列表和规则集。

❓ 速度很慢怎么办？

优化建议：

  1. 尝试通过节点测试，手动切换到延迟更低的节点
  2. 检查本地网络是否正常
